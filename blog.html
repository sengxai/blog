CS268: Web Systems: Blog Entry 1

For my blog entry, I chose to look at navigation bars from w3schools. It is mainly based upon CSS style of how I would like to design the navigation to be veritcal or horizontal style. This site shows different types of CSS navigational bars. By learning how to do navigational bars on my own webpage it would be more easier to navigate through my webpages more easily.

From this site I was able to learn about the basics of how to make a navigational bar. I've learnt how to keep active color in the CSS stylesheet so I could tell what webpage I am currently on. I've also learnt how to make a basic horizontal and veritcal navigation bar with unorder list <ul> and <li>.

Example Code

            <head>
            <style>
            ul {
            list-style-type: none;
            margin: 0;
            padding: 0;
            overflow: hidden;
            background-color: #333;
            }

            li {
            float: left;
            }

            li a {
            display: block;
            color: white;
            text-align: center;
            padding: 14px 16px;
            text-decoration: none;
            }

            li a:hover {
            background-color: #111;
            }
            </style>
            </head>
            <body>

            <ul>
            <li><a class="active" href="#home">Home</a></li>
            <li><a href="#news">News</a></li>
            <li><a href="#contact">Contact</a></li>
            <li><a href="#about">About</a></li>
            </ul>

            </body>
            </html>
